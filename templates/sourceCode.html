Source Code and System


UCI Heart Disease Dataset

The dataset used in our project was aggregated from four different datasets collected from Hungary, Switzerland, and the US. Percentage of aggregated data obtained from each source is shown in the adjacent pie chart. The data itself was available in a text file with each row broken up across several lines. Using a parser script we wrote in python, we aggregated data from these sources and stored it in csv format. After aggregation, we had 899 records with 76 attributes. However, after initial exploration, we got rid of 26 variables which contained the same value across all records of the dataset. These variables included ssn, name, etc. pertaining to individual characteristics of the corresponding patients, which were anonymized by adding dummy values in their place.
