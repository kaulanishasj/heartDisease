Data Sources

We used two kinds of data sets to answer the two questions we formed. In order to answer the question on "How prevalent is Heart Disease in USA?" we used fact sheets published by Center for Disease Control and Prevention (CDC) which can be found here. Various statistics about heart disease prevalance are regularly collected and maintained by CDC. Hence this dataset seemed to be right choice to talk about the prevalance of this disease.

In order to answer the question on "What may be the causes of heart disease?" and "What are my chances of being suffering from heart disease?" We used dataset published on UCI data repository. This dataset contained various characteristics of patients such as their cholestrol level, whether they had chest pain, age, sex, etc. along with whether they were diagnosed with a heart disease. Hence we used this dataset to provide insights on what may be the causes of heart diseases using the information provided for diagnosed patients.

The datasets used for exploration from CDC did not require any processing. The data was very clean and coherent with no outliers. Hence we only discussed the UCI heart disease dataset in the following sections which have interesting data quality characteristics.

UCI Heart Disease Dataset

The dataset used in our project was aggregated from four different datasets collected from Hungary, Switzerland, and the US. Percentage of aggregated data obtained from each source is shown in the adjacent pie chart. The data itself was available in a text file with each row broken up across several lines. Using a parser script we wrote in python, we aggregated data from these sources and stored it in csv format. After aggregation, we had 899 records with 76 attributes. However, after initial exploration, we got rid of 26 variables which contained the same value across all records of the dataset.

Data Transformation

The target labels of this data initially ranged from 0 to 4, where 0 implied no presence of heart disease. It was not clear what values 1 through represent from the data representation, but they all denoted the presence of some form of heart disease. Hence we transformed 1 through 4 values to 1, so that 0 implies no heart disease and 1 implies presence of heart disease.
